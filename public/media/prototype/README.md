# Prototype media

Asset cục bộ chỉ phục vụ QA bố cục. Production content và `src/content/assets.ts` không được tham chiếu file trong thư mục này.

Khi semantic asset chưa sẵn sàng, giữ record manifest không có `src`; runtime tự hiển thị slot lưới/linear tĩnh đúng tỷ lệ, không thêm ảnh giả hoặc chữ placeholder.
