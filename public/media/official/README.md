# Official media

Chỉ đặt asset AIC/HaUI đã được xác minh và phê duyệt tại đây. Runtime tham chiếu file bằng semantic ID trong `src/content/assets.ts`; không sửa JSX để thay media.

Định dạng được duyệt: SVG cho logo; WebP cho ảnh/poster; WebM hoặc MP4 cho video. Tỷ lệ chuẩn: hero/card 4:3, video/map 16:9, portrait 4:5, partner logo 3:2 và brand logo 1:1.

- `aic-logo.jpg`: logo AIC hiện tại, khai báo bằng `brand.aic.logo`.
