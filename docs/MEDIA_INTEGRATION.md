# Media Integration

## Locations

- `public/media/official/`: asset đã duyệt dùng production.
- `public/media/prototype/`: asset thử nghiệm cục bộ, không dùng làm dữ liệu chính thức.

Tham chiếu asset bằng path bắt đầu từ `/media/`, ví dụ `/media/official/hero.webp`. Không import URL ngoài repo.

## Formats

- Hero: WebP 1600–2200px hoặc WebM ngắn, tắt tiếng.
- Card: WebP 800–1200px.
- Portrait: WebP 600–900px.
- Video 03:20: WebM/MP4 tối ưu và poster WebP.
- Logo: SVG/WebP chính thức, giữ đúng tỷ lệ.

Mọi asset có ý nghĩa phải có `alt`. Không dùng chân dung AI, stock ngẫu nhiên hoặc logo tự tạo.
