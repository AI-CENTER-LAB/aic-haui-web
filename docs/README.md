# Project documentation

`DESIGN.md` và `CODE_PLAN.md` ở root là nguồn chuẩn.

- `CURRENT_STATE_AUDIT.md`: baseline, asset, route và lỗi trước rebuild.
- `CONTENT_INTEGRATION.md`: data schema, quy trình xác minh và cách bật section.
- `MEDIA_INTEGRATION.md`: asset path, kích thước, API media và cách thay media sau này.
- `QA_REPORT.md`: test/build/Docker/Lighthouse và responsive evidence của bản bàn giao.
